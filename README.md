# Abdul Sami — Portfolio

Personal portfolio website built with React, TypeScript, and Tailwind CSS.

**Live:** [abdulsami.app](https://abdulsami.app)
**GitHub:** [github.com/abdulsami-1/abdulsami-portfolio](https://github.com/abdulsami-1/abdulsami-portfolio)

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite
- EmailJS (contact form)

## Getting Started

```bash
npm install
npm run dev
```

## Contact Form

Powered by EmailJS. To use locally, replace the service ID, template ID, and public key in:

- `index.html` — public key (`emailjs.init(...)`)
- `src/components/Contact.tsx` — service ID and template ID

Create your own free account at [emailjs.com](https://www.emailjs.com).
