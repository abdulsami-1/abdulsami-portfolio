## Contact Form

Powered by EmailJS. To use locally, replace the service ID, template ID, and public key in:

- `index.html` — public key (`emailjs.init(...)`)
- `src/components/Contact.tsx` — service ID and template ID

Create your own free account at [emailjs.com](https://www.emailjs.com).
